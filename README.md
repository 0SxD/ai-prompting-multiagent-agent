# ai-prompting-multiagent-agent

Research scaffold for multi-agent prompt orchestration patterns and prompting methodology.

## Status

R&D / Experimental. Maintained by Sage / 0SxD as part of an ongoing prompt-engineering and agent-skills research portfolio.

## What this is

An agent architecture scaffold exploring prompting strategies for multi-agent systems. The repo organizes a reading corpus across three domains - prompt engineering, multi-agent systems, and LLM orchestration - and seeds a set of skills for extracting reusable prompting patterns. Focus is on the research and methodology layer, not production deployment.

## Layout

- `spec.md` - session state and hot-cache protocol; read first per agent entry
- `system_directive.md` - agent operating directive and role constraints
- `bootstrap.md` - boot sequence and session initialization instructions
- `skills.md` - skill registry (seeded; drafts in progress)
- `metrics_ledger.md` - cost and progress ledger
- `corpus/` - organized reading corpus (prompt_engineering, multi_agent_systems, llm_orchestration sub-folders)
- `Sage_Agent_Prompt_Engineer_bundle_v1.zip` - bundled prompt-engineer skill set (see spec.md for ingest plan)
